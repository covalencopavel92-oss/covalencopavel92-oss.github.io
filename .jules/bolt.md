## 2024-05-16 - Safe Event Loop Yielding in Isomorphic Environments
**Learning:** When adding event loop yields (like `requestIdleCallback`) to optimize client-side parsing (e.g., in `.astro` files), blindly referencing `window` will throw `ReferenceError: window is not defined` during Server-Side Rendering (SSR) or build time.
**Action:** Always guard browser globals in universally rendered components. Use `typeof window !== 'undefined' && 'requestIdleCallback' in window` instead of just `'requestIdleCallback' in window` when implementing client-side performance optimizations in frameworks like Astro.
